Audio dictionary for Nufi

About the Project

This dictionary aims to preserve and revitalize the Nufi language by providing users with a platform to search, hear, and learn words effectively. The app is hosted on Firebase Hosting and integrates with Firestore for real-time data management.

Features

  • Interactive Search Bar: Allows users to search for words and view real-time suggestions.
  • Word Details: Provides word definitions, phonetic pronunciations, and audio playback.
  • Custom Encoding: Applied Clafrica mapping to handle complex linguistic transformations.
  • Environment-Specific Data Loading: Utilized local data during development and Firebase in production.

Technologies

  • Front-End Development: Built the user interface using React.js, ensuring responsiveness and interactivity.
  • Firebase Integration: Set up and configured Firestore for real-time database management and deployed the app using Firebase Hosting.
  • Data Transformation: Designed a robust mapping system using JavaScript to handle Clafrica encoding for linguistic data processing.
  • Full-Stack Development: Worked on both client-side and server-side logic for smooth app functionality.
  • Search Optimization: Implemented an efficient search algorithm to filter and display words dynamically.
  • State Management: Used React hooks (useState and useEffect) for seamless data handling.
  • UI/UX Design: Designed a clean and intuitive user interface with Tailwind CSS.
  • Problem Solving: Resolved complex issues, such as real-time data syncing and conditional environment setup for development and production.
